# Vertical Data Sources Catalog

> **TL;DR**: Source catalog the [[../methodology/investment-thesis-protocol|investment-thesis protocol]] §5 (money-flow validation) cites. Vertical-scoped to portfolio-relevant verticals: crypto (active), real estate + food/commodities (adjacent — Farmers Financing context), tech-AI (broad investing landscape). Defense / healthcare cite-only as future scope. Each entry annotated with what it covers; gap flags surface coverage holes.

## The problem this page addresses

Money-flow-audit arm needs vertical-specific sources to confirm capital is moving with the thesis. Without an anchor, sources get fabricated or generically cited (violating worldview prior #7). This page is the substrate the arm reads.

This catalog **names** sources. It does not assert what those sources currently report — that's per-thesis work, with episodes cited at the §4 historical-analog field.

## Crypto (active)

Most flow data already lives in trading branch foundations. Re-indexed here from the investment angle.

- **On-chain flows** — Glassnode, CryptoQuant, Nansen, Arkham, on-chain explorers (Etherscan, Solscan)
- **Exchange data** — Coinglass (liquidations, OI, funding), Velo, Laevitas, Skew
- **ETF disclosures** — issuer 8-K filings (iShares, Fidelity, ARK), SEC 13F for institutional ETF holders
- **Fund letters** — Pantera, Multicoin, Paradigm, a16z crypto, Polychain
- **Macro / policy** — FOMC dot plot, US Treasury rates, ETF flow reports, executive orders
- **Reports** — Galaxy Digital research, Coinbase Institutional, Glassnode Insights

## Gap flags for this catalog

- **Smart-money positioning beyond 13Fs** is hard to source for crypto (no equivalent of 13F filings) — ETF holder disclosures + fund letters are partial substitute
- **Real estate granularity below city level** requires paid services (CoStar, Reonomy)
- **Tech-AI primary sources** are public but narrative timing requires Stratechery / SemiAnalysis paid tier
- **Food / commodities forward-looking flows** (capex, planted acreage) lag actual reports
- **Crypto fund letter access** is uneven — fund-by-fund quality varies; no central registry

## How to use

In §5 of an investment-thesis artifact:
- Cite specific sources from this page that cover the question
- If a question isn't covered by any listed source — flag in §5's gap flags, do not fabricate
- If money-flow-audit recurringly hits a gap → propose addition via wiki-refinement arm

## Key Takeaways

- Vertical-scoped to user portfolio; defense / healthcare cite-only
- Sources organized by category: government / institutional / corporate / smart money / reports / ETFs
- Money-flow-audit cites from this page; flags gaps when a question isn't covered
- Catalog grows when an actual thesis surfaces a missing source
